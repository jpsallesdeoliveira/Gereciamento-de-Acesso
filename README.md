# Gereciamento de Acesso

O projeto **Gereciamento de Acesso** tem como objetivo gerenciar o acesso dos membros à 2 salas da empresa. Utilizando o ESP32 DevKit V1 e a plataforma Arduino, o sistema proporciona um controle de acesso seguro e eficiente, facilitando o monitoramento e a gestão das entradas e saídas.

## Funcionalidades

- **Controle de Acesso:** Monitoramento de entradas e saídas dos colaboradores em 2 salas.

## Tecnologias Utilizadas

- **Hardware:** ESP32 DevKit V1
- **Plataforma de Desenvolvimento:** Arduino IDE
- **Linguagem:** C/C++ (Arduino)
- **Comunicação:** Wi-Fi (Web Server)

## Requisitos

- ESP32 DevKit V1
- Cabo USB para conexão com o computador
- Arduino IDE com suporte para ESP32
